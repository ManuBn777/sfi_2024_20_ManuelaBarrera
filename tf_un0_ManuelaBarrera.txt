b) Que  diferencia hay entre repositorio local y remoto?
R// Repositorio Local: Este se encuentra en la pc/computador
Repositorio Remoto: Por otro lado el remoto se encuentra en la internet/nube

c) Mencione tres diferencias entre Git y GitHub
Git:
- Es un repositorio local
- Se instala en el computador/local
- Esta funciona con líneas de códigos y una consola
GitHub:
- Es un repositorio remoto
- Se usa en la web y requiere internet
- Esta tiene interfaz grafica

d) Área de trabajo, área de preparación (ubicarse para la foto) y directorio Git (toma de la foto)
Podrías explicar en que consisten?
R//
Área de trabajo: Es la ubicación del pc donde se encuentran tus archivos del proyecto
Área de preparación: Es cuando lo añades (Git add) y esta listo para ser registrado <-- (tomarle la foto)
Directorio Git: Es cuando tomas la foto (Git commit) y guardas la versión del archivo
