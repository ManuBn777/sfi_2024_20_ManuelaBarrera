R// Repositorio Local: Este se encuentra en la pc/computador
Repositorio Remoto: Por otro lado el remoto se encuentra en la internet/nube

c) Mencione tres diferencias entre Git y GitHub
Git:
- Es un repositorio local
- Se instala en el computador/local
- Esta funciona con líneas de códigos y una consola
GitHub:
- Es un repositorio remoto
- Se usa en la web y requiere internet
- Esta tiene interfaz grafica
